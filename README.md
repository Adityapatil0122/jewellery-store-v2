# jewellery-store-v2

Simple modern jewellery catalogue website (static HTML/CSS/JS) with product cards, pricing, and social sharing.

## Run locally

```bash
python -m http.server 4173
```

Open `http://localhost:4173/index.html`.

## Features
- Jewellery listing with search and category filters
- Product cards with image, price, and short description
- Sharing actions (WhatsApp, copy link, native share, Instagram shortcut)
- Lightweight static-first setup for easy free hosting
